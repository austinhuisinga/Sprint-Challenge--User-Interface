

1. If you were to describe semantic HTML to the next cohort of students, what would you say?

  Semantic HTML is used to give code meaning. It both helps make code more legible for yourself, others, and screen readers, and also enhances SEO.

2. Name two big differences between display: block; and display: inline;.

  Block elements take up the full width available on a given page, while inline elements only take up as little space as needed. Block elements will start a new line every time, inline elements will stay in the current line. 

3. What are the 4 areas of the box model?

  Margin, Border, Padding, Content. 

4. While using flexbox, what axis does the following property work on: align-items: center?

  Cross axis. 

5. Explain why git is valuable to a team of developers.

  It lets teams work with the same files, and know what everyone else is working on in (more or less) real time.
